      肆人物檔案

      伍對話與預測

        造一個新人物

        若語料是多人對話(群組聊天、訪談逐字稿),填上此人在紀錄裡的稱呼,蒸餾時只會鎖定他的發言;別人對他的評價則歸入「他者視角」。

        設定

        走 Anthropic Messages API 相容端點,重用雲端串流邏輯。MiniMax 模型:MiniMax-M3(1M 上下文,蒸餾建議)、MiniMax-M2.7 等。MiniMax 預設吐簡體,建議搭配下方「強制繁體輸出」。

        ⚠ 本地小模型的蒸餾品質(尤其心智模型提煉)明顯不如雲端 Opus;建議搭配快速檔、或大參數量的長上下文模型。對話模式則較友善。語料上限已自動下修為 12 萬字。

        維度分析是吃 token 的機械式抽取,可指定便宜模型;真正難的綜合提煉與 persona 組裝仍用上方主模型。

        蒸餾大陸講者時,「只轉字不改詞」能保留他真實的用語習慣(表達DNA),不被台灣化。
